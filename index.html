Dynamic Memory Allocator

I completed this project while taking ICS 53 at UC Irvine. The project is an implementation of memory allocator akin to malloc (libc). The allocator utilizes an explicit free list to satisfy dynamic memory allocation requests. It implements malloc, realloc, calloc, and free routines.

Operating System Process & Resource Manager

I completed this project while taking CS 143A at UC Irvine. It is a simulation of a uniprocessor system with processes and resources (e.g., semaphores, files, etc). Additionally, a user can create processes with one of three priorities. The system then schedules these process according to priorities. Each process can request a portion of the multiunit resources available.
